This is a [Next.js](https://nextjs.org/) project bootstrapped with [`create-next-app`](https://github.com/vercel/next.js/tree/canary/packages/create-next-app).

## Getting Started

First, install dependencies:

```bash
npm install
# or
yarn 
```

Then, run the project:
```bash
npm run dev
# or
yarn dev
```

To run unit tests:
```bash
npm run test
# or
yarn test
```
